# Strategy Kit Instructions

This is the Admireworks strategy-and-presentation system for new client projects.

## Purpose
Use this kit after the client KB has enough source material to build a serious strategy draft.

## Inputs
Before using this kit, make sure the project already contains:
- a populated `docs/client_kb/INDEX.md`
- at least one brief
- at least one structured meeting note or raw transcript
- any brand, offer, or campaign constraints recorded in `docs/project/PROJECT_CONTEXT.md`

## Workflow
1. Read `strategy-kit/STRATEGY-SYSTEM.md`.
2. Read the project context, the client KB, and the research layer.
3. Generate the strategic outputs into `docs/strategy/`.
4. Only after the written strategy is reviewed and approved, generate the presentation build in `apps/presentation/`.
5. Use docs/strategy/presentation-outline.md to map each slide back to the strategy docs.

## Rule
Do not invent strategy in a vacuum. The source material in `docs/client_kb/` must drive the result.
